使用Hexo快速搭建网站页面

步骤

1. 下载Hexo

npm install hexo-cli 下载到本项目，若是下载到全局内 npm -g install hexo-cli.

2. 问题

  • hexo init 执行需要在空目录下。如果你是下载到了本项目，就在项目下创建个新文件夹，然后进入到新文件夹内执行。

  • 如果hexo-cli 安装了但是hexo命令找不到，可以尝试使用 npx hexo 形式执行。
