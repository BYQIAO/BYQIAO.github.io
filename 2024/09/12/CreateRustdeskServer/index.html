    环境

    ubuntu

    步骤

    先查看下系统环境，然后下载对应的版本

    • arch

    下载rustdesk-server

    • wget https://github.com/rustdesk/rustdesk-server/releases/download/1.1.11-1/rustdesk-server-linux-amd64.zip

    • unzip <FileName>.zip 解压刚才下载的zip压缩文件，里面有hbbr和hbbs

    • 使用官方推荐的pm2管理服务。官方参考文档

    安装nodejs环境

    • 选择符合自己系统架构的版本arch 可查看架构，下载编译好的二进制文件Prebuilt Binaries ,下载是一个 .tar.xz 压缩文件

    • tar -xf 解压

    • vim ~/.bashrc 编辑环境变量文件，然后添加export PATH=$PATH:[nodejs的可执行文件夹的路径]

    • 使用npm -v 和 node -v 查看是否可用

    安装pm2 及使用

    • npm config set registry https://registry.npmmirror.com 换成国内源

    • npm install pm2 -g [-g]就是全局安装

    • pm2 start hbbs -- -r [服务运行的主机IP地址,你在哪台机器上搭建服务，你就填那个主机的IP] 如果是默认的21117端口，则IP后面可以不填写[:port]

    • pm2 start hbbr

    • pm2 list 可以查看hbbs和hbbr状态

    • 设置重启后自动运行pm2 save 和pm2 startup

    • hbbs -r 参数可以不使用。使用-r 可以让你在rustdesk软件内配置时不填写中继服务器

    设置使用的端口

    21116-21119端口都打开tcp，21116还要开启udp

    • sudo ufw allow 21116/tcp 按照此格式开启要使用的端口和功能

    • sudo ufw allow ssh 允许ssh连接

    • 你还要注意开启你其他服务需要使用的端口

    • sudo ufw enable 启动防火墙

    关闭ufw也时可以运行服务的。
