Linux下搭建mqtt服务器，使用mosquitto

环境

  • ubuntu

步骤

添加仓库

sudo apt-add-repository ppa:mosquitto-dev/mosquitto-ppa 现在好像不需要也行。

导入

sudo apt update

sudo apt install mosquitto

sudo apt install mosquitto-dev

添加配置文件

在/etc/mosquitto/conf.d 目录下创建.conf 文件。

在此文件内添加:

  • listener 1883 （监听1883端口）

  • allow_anonymous true 允许匿名访问

设置mosquitto服务

  • sudo service mosquitto restart 重启下服务

  • sudo service mosquitto status 查看服务是否启动

  • sudo systemctl enable mosquitto 设置服务开机自启
